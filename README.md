# Bank Management System OOP

## Installation

1. Clone the repository to your local machine.
2. Compile the source files using your preferred C++ compiler.
3. Run the compiled executable to start the Bank Management System.

## Usage

1. The system will prompt you to log in. Enter a valid username and password.
2. Once logged in, you will see the main menu with the following options:
   - Find Client
   - Update Client
   - Add New Client
   - Delete Client
   - Client List
   - Transactions
   - Manage Users
   - Registers
   - Logout
3. Select an option by entering the corresponding number.
4. Follow the on-screen instructions to perform the desired actions.


## Testing

The Bank Management System includes the following test cases:

1. Validate user login with correct and incorrect credentials.
2. Test the functionality of adding, updating, deleting, and finding clients.
3. Verify the accuracy of client account balance calculations and transactions.
4. Ensure the user management features (add, update, delete, and find users) work as expected.
5. Test the access control system to ensure users can only perform actions based on their permissions.

To run the test cases, compile the source files and execute the compiled program.
